# RIURSA — Sitio institucional

Rediseño de la página principal de la Red Internacional Universitaria de Responsabilidad Social y Ambiental (RIURSA) en Perú. El sitio presenta la misión de la organización, universidades adheridas, actividades, publicaciones, marco normativo y vías de contacto en una experiencia moderna y accesible.

## Tecnologías

- HTML5 semántico
- CSS3 con variables, Grid, Flexbox y animaciones
- JavaScript nativo para navegación móvil, scroll spy y revelado de contenido
- Google Fonts (`Libre Franklin` y `DM Sans`)

## Ejecutar localmente

No requiere instalación ni proceso de compilación. Puede abrirse directamente `index.html` en un navegador o servirse con cualquier servidor estático.

```bash
npx serve .
```

La paleta, tipografías, radios, sombras y medidas principales se centralizan en las variables de `styles.css`.
